# Data Center Incident Analysis Dashboard

## Overview
Interactive dashboard analysing data center server incidents and downtime patterns built with Tableau Public.

## Dashboard
🔗 [View Live Dashboard](https://public.tableau.com/app/profile/praveena.seeniwasaham/viz/DCIncidentAnalysisDashboard/Dashboard1)

## What the Dashboard Shows
- Incidents by Type (Power Failure, Hardware Fault, Network Outage, Cooling Issue)
- Total Downtime by Server
- Incidents by Technician
- Incidents by Severity

## Tools Used
- Tableau Public
- Data source: dc_incidents_cleaned.csv (cleaned using Python/pandas)
